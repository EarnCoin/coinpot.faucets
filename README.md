# coinpot.faucets
<p>它是一個MicroWallet。MicroWallet大部份都為水龍頭的網站在使用，目的是儲存小額虛擬貨幣，您可以透過這個Microwallet再把虛擬貨幣轉到交易所換成現金（這個之後會寫文章再做介紹）或把虛擬貨幣匯到其他錢包去做為不同的用途。所以大家在開始賺取水龍金的虛擬貨幣時，一定要先來注冊Coinpot，以下為Coinpot的操作，希望可以幫助大家更快了解CointPot的使用方法。</p>
<p>Coinpot是2017成立的公司，它是一個整合多家水龍頭網站MicroWallet，目前支援的水龍頭網站包含：Moon系統水龍頭、Bonusbitcoin,和Bitfun。作為一個MicroWallet錢包，他支持儲存比特幣、Litecoin 和 Dogecoin 貨幣。</p>

<h2>建立虛擬貨幣（BitCoin、LiteCoin、DogeCoin）錢包地址</h2>
<p>Coinpot是一個MicroWallet，所以可以提供使用者從外部其他來源（如：交易所、錢包等）存入BitCoin、Litecoin、DogeCoin等虛擬貨幣。因為從外部其他來源（如：交易所、錢包等）需要有一個錢包地址，因此大家若要利用Coinpot收取其他來源的款項時，一定得要在各虛擬貨幣頁面各別產生一個錢包地址（請點選Deposit Bitcoin- ＞Generate Bitcoin ），Coinpot就會產生一組亂碼式的字串，這個就是您的錢包。這就有點類似，你去銀行開存款帳戶，行員幫您完成開戶手續後，會給您一組帳號，日後，有人要匯款給您，你就會把這個帳號給對方，對方就會把錢匯到您的帳戶去。如下圖：</p>

<h2>虛擬貨幣出金 / 轉出</h2>
<p>這個當然就是大家最關心的部份，我們在各水龍頭所賺取的虛擬貨幣要如何匯出去？那您就一定要了解這個功能了。這個功能就有點像大家使用網站銀行把帳戶裡的錢匯款至其他帳戶去。您在moonbit、moonliteco、moondoge、Bonusbitcoin和Bitfun所以賺取的水龍頭虛擬貨幣會直接存入到Coinpot，您可以直接在Dashboard看到匯入紀錄。</p>
<p>目前Coinpot提供的比特幣的提款方式: FaucetHub-最低提款10k satoshi (1.5% 费) FaucetSystem-最低提款10ksatoshi(2% 费) Xapo-最低提款10k satoshi (不收费) 比特币地址-最低提款10k satoshi (1000 satoshi支付的費用, 低于50k satoshi, 更大的付款是免費的) 所有的付款是在48小時内完成出金。針對LiteCoin及DogeCoin都有明確標明匯出的手續費。</p>
<p>要執行虛擬貨幣出金的操作很簡單，以下會以Bitcoin作為案例，請參照以下的步驟進行即可：</p>
<ol>
<li>連結至：<a href="https://coinpot.co/" target="_blank" rel="noopener">Coinpot </a>並點選Sign In按鈕</li>
<li>點選Bitcoin</li>
<li>點選Withdraw Bitcoin</li>
<li>在Withdrawal method欄位往下拉，就可以選取您要轉出目的地錢包，這有如您在使用網路銀行轉帳時，您首先要所選擇要轉出的銀行。</li>
<li>在藍色的區塊都有寫清楚在何種條件之下，可以享有免手續費</li>
<li>在Withdrawal <span data-bind="text: currencySummary().currency().toLowerCase()">bitcoin</span> address欄位，輸入您要轉出的目的地，這的意思就像是您在使用網路銀行轉帳給對方時，您要輸入匯款對像的帳號。</li>
<li>在Amount to withdraw欄位，輸入您要轉出的金額，這的意思就像是您在使用網路銀行轉帳給對方時，您要輸入匯款金額。</li>
<li>在Password欄位，重新再輸您在Coinpot的密碼，這也是像在網路銀行在轉帳時，一樣要重新再做一次安全驗證</li>
<li>點選Withdraw按鈕</li>
<li>在數分鐘後，您在Coinpot注冊的EMAIL會收到一封確認轉出的信件，請務必要點選信件內的連結，不然這一筆匯款資料不會成功被執行哦</li>
<li>48小時內會收到貨幣</li>
</ol>

<h2>轉換虛擬貨幣</h2>
<p>這個功能就像是，我們在銀行將新台幣轉換成美金或日幣的功能。在Coinpot，目前提供三種幣別互相轉換，如：您可以將DogeCoin轉成Bitcoin或將BitCoin轉換成Litecoin. 目前Coinpot平台上做幣別的轉換都是免手續費的。轉換幣別的操作與出金的方式類似。</p>
